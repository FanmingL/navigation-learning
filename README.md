# navigation-learning
## Dependencies
* Ubuntu 16.04
* [ROS Kinetic](http://wiki.ros.org/kinetic/Installation/Ubuntu)
* Pybullet
* Python2.7
* Google Protobuf

```bash
sudo apt-get install protobuf-compiler libprotobuf-dev cmake3
sudo pip2 install pybullet numpy
```
